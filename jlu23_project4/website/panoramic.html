2. Panoramic transformations

In this section, we compute the sphere normal using the pixel location and basic geometry. Then using constant V as (0,0,1) and the formula R = V - 2 .* dot(V,N) we can calculate the reflection vector. We use the reflection vector to calculate the phis and thetas of pixels. The correspondance between sphecial coordinates and image value is built using the interpolation function TriScatteredInterp(). Then the final result for the equirectangular domain is computed using the mapping to sperical coordinates's correspoinding pixel value.
